## Getting started
### Export your bot token in your `~/.bashrc` file
```sh
export BOT_TOKEN=your_token_here
```

### Install dependencies
```sh
npm install
```

### Start the bot
```sh
npm start
```
## Development
### Recommanded IDE Extensions
* `ESLint` Linting support
* `EditorConfig` Standardized editor configuration

### Run linter manually
```sh
npm run lint
```

### Run unit tests
```sh
npm test
```
